Student Grade Analyzer

A command-line C program developed as part of the ICP Practical assignments.
The application takes student marks for Mathematics, Science, and English to calculate total scores,
overall averages, final letter grades, and pass/fail eligibility based on specific academic criteria

Grading Scale

80-100 : A
70-79 : B
60-69 : C
50-59 : D
Below 50 : F

Pass / Fail Condition

Pass : The student must score 40 or above in ALL three individual subjects.
Fail : If any individual subject score falls below 40, the student fails the entire term (regardless of how high their average is).

Distinction Bonus

if the overall average is 85 or above AND all individual subject marks are 80 or above.
